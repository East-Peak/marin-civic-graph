# data/raw/

Raw source captures (PDFs, OCR snapshots, archived HTML, file-by-file manifests) live here when the ingestion pipelines run.

These artifacts are **not in this repository's git history**. They were removed via `git filter-repo` to keep clone size manageable (the captured set was ~200 MB of public-record agenda packets and FPPC export PDFs). They remain on the author's local machine and can be regenerated by re-running the relevant capture script under `scripts/capture_*.py`.

Each capture run writes a sibling manifest file describing what was pulled, with source URLs and content hashes. The downstream extraction pipelines (`scripts/extract_*.py`) read from here; the normalized bundles in `data/normalized/` reference back via record IDs.

For source-method documentation, see [`/docs/source-inventory.md`](../../docs/source-inventory.md), [`/docs/jurisdiction-source-map.md`](../../docs/jurisdiction-source-map.md), and the per-source bundles under `docs/*-source-bundle.md`.
